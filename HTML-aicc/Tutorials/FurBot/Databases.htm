FurBot Member Database

[1] Add an Action to your bot and name it 'Create DB'. Change the If Statement to look like the following:

If * Whispers me CreateDB
Add a statement to the Then box and select:
$DATABASE: Create Table #1 to database #2
:from the pulldown menu. In the first small textbox type 'Members' and in the second textbox type '1'. Click Update, click update again.

[2] Add another Action. Name it 'Add Member'. Change the If statement to look like the following:

If * Whispers me join
This action will be triggered when any furre whispers the bot 'join'. Add a statement to the And box, select:
$DATABASE: if there is not a record in table # in database # where id is #
:from the pulldown menu, type 'Members' in the first textbox, '1' in the second textbox, and '[player]' in the third textbox, and click Update. Add a statement to the Then box and select:
$DATABASE: Add record where #1 value is #2 to table #3 database #4
:from the pulldown menu. Type 'ID' in the first textbox, '[player]' in the second textbox, 'Members' in the third textbox, and '1' in the fourth textbox. Click Update, click Update again, and add another statement to the Then box. Select 'Whisper back #1' from the pulldown menu and type 'You are now a member.' in the small textbox. Click Update, click Update again.

[3] Add another Action. Add a Trigger and name it 'OpenDB'. Leave "When Connection establishes" as it is. Add a statement to the Then box. Select:

$DATABASE: Open database #1 to ID #2 (1-4)
:from the pulldown menu. Type member.mdb in the first text box and 1 in the second text box. Click the Update button.

[4]Add another trigger and name it 'Close DB'. Select:

Connection closes
:from the pulldown menu. Add a statment to the Then box and select:
$DATABASE: Close database #1
:from the pulldown menu. In the small text box type '1'. Click Update, click Update again, Close, and Save.

[5]Name this action 'Remove Member'. Change the If statement to look like the following:

If * Whispers me retire
Add a statement to the And box and select:
$DATABASE: if there is a record in table # in database # where ID is #
:from the pulldown menu. Type 'Members' in the first textbox, '1' in the second textbox, and '[player]' in the third textbox. Click Update, and add a statement to the Then box.

[6]Select:

$DATABASE: Remove record from database #1 table #2 where ID is #3
:from the pulldown menu. Type '1' in the first textbox, 'Members' in the second textbox, and '[player]' in the third textbox. Click Update and add another statement to the Then box. Select:
Whisper back #1
:from the pulldown menu and type 'You are no longer a member.' in the small textbox. Click Update, click Update again, and Close.

[7] This next Action is optional but useful. Add an Action and name it 'List Members'. Change the If statement to look like the following:

If * Whispers me list members
Add a statment to the Then box and select:
$DATABASE: List all IDs from database #1 table #2 which start with #3 to variable #4
Type '1' in the first textbox, 'Members' in the second textbox, the third textbos can stay empty, and '[tmpList]' in the fourth textbox. Click Update, click Update again, Close, and Save. Now you have a member database. Members can join by whispering the bot 'join', quit by whispering the bot 'retire', or anyfurre can get a current list of members by whispering the bot 'list members'.
