DS Responses With FurBot

The first thing you'll need to do is 'show' the DS on the bots screen. To do this make a Trigger and name it 'Show DS'. In the 'When' statement choose 'DS-trigger # is sent' from the pulldown menu.

Click the Add button under the Then box. Choose Add text #1 to the connection log. In the small textbox type: DS: dx: [dsx] dy: [dsy] dx2: [dsx2] dy2: [dsy2] did: [dsid] didx: [dsidx] didy: [dsidy]
Click Update, then Update on the Trigger windows, Close the Trigger window, and save your bot.


About DS Variables: 

[dsx] - Contains the previous X Coordinate of the triggering furre.
[dsy] - Contains the previous X Coordinate of the triggering furre
[dsx2] - Contains the X Coordinate of the trigger
[dsy2] - Contains the Y Coordinate of the trigger
[dsid] - Contains the trigger number
[dsidx] - Contains the new X Coordinate of the triggering furre.
[dsidy] - Contains the new Y Coordinate of the triggering furre.

Trigger Summary:  
When DS is triggered, the triggering furre's previous and new position, the trigger position, and the trigger number are all shown in the bots connection log.
