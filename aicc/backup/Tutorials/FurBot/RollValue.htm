Store The Value of a Roll

Create a trigger and name it Store Roll Value.

The When statement should look like th following: When Furcadia sends command # [#] * rolls * gets *.

Add a Then statement and select Store #1 to variable #2.

In the first textbox type [word3]. In the second textbox type [tmpRoll]. Click Update. Clik Update again. Click Close. Save your bot.

Tip: Add another Then statement that has the bot emit the value of [tmpRoll] so that you know the trigger is successful. You can name the variable you're using to store the die roll anything you like.

Extras: You can add an And statement to check to see if they furson rolling is a certain furson. Add this And statement to do that: variable [tmpFtr1] is [word1]. You will have to have stored a fursons name to the variable [tmpFtr1] before this trigger is set off for this to work.
