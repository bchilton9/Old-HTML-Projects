Install and use of Teamspeak for Freefire

  • Download the program files from http://www.teamspeak.org.
  • You will see the Quick download box in the upper right corner of the page.
  • You need to download the one marked "TS2 Client (Win)".
  • Save it to your hard drive then install the program.

  • When you have it installed, run the program you will see a window like this:

  • You now need to add a Push to talk button.
  • Select Settings > Sound Input/Output Settings.

  • In the Sound Settings under Voice Send Method select Push to Talk.
  • Then press the "Set" button. Youll get a message saying "Press a key" press the key you wish to use as your push to talk button. This key will need to be held down when talking. Most people are useing the "End" key.
  • Press "Close"

  • Select Connection > Connect

  • You will see the server select screen.

  • Right click on "Servers" and select "Add Server".

  • Label the server "Freefire"
  • Put in the "Server Address:" of "freefire.game-server.cc:8777".
  • Put in a "Nickname:" of your player name.
  • Select "Anonymous",
  • Put in the "Password:" of "freefire42" (case sensatice).
  • Select "Auto-Reconnect"
  • Then click "Connect".

  • You will now me connected to the server as an Unregistered User. You can still talk to everyone!
  • Look on the list of members connected for one that has a "(R SA)" next to there name and ask them to flag you to register. If there is no one with "(R SA)" next to there name you will have to wate to register.

  • Once you have been flaged you can register a login name and password.
  • Select Self > Register With Server.

  • You will see the Register window.
  • Put in a "Login Name:" of your choice.
  • Put in a "Password:" of your choice.
  • ReType in your password in the "Password Check:".
  • Make a note of the Login Name: and Password you use and rember theay ARE case sensative.
  • Click on "Register".

  • You are now Registered with the server but you are not done yet.
  • Disconect from the server by going to Connection > Disconnect.

  • Select Connection > Connect once more.

  • On the Server Window make the following changes.
  • Change "Anonymous" to "Registered".
  • Put in the "Login Name:" and "User Password:" you registered with.
  • Click "Connect".

You are now registered and set to use teamspeak! If you have any problems please speek to one of the officers in game or on the website.

If you are an Officer ask a "(R SA)" to flag you as an Officer.
