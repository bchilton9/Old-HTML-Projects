What is Chicken Prox and what dose it do?
Chicken Prox is a Proxy server. It will add features to Furcadia that are not currently there.
Unlike most proxy's for Furcadia, Chicken Prox is 100% on your computer.
You don't have to connect to a proxy on someone else computer then to the Furcadia server.
When you start Chicken Prox, It will check for any updates, if there is any it will download them and install them.
If there is no updates Chicken Prox will place its self on your task menu and launch Furcadia.
Then when you are in game, Type #HELP for commands to use your new features.
